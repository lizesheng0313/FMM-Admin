## 前言

`FMM-Admin`是一个 SAAS 电商后台管理系统，界面优雅，功能丰富，小巧迅速，包含一个电商平台完整功能，具备良好的审美风格与编码设计。一套代码支持多商城管理

## 项目介绍

### 使用技术

- Vue3
- ElementPlus

- Axios
- Vue-Router

### 功能

- 登录 JWT 认证
- 会员中心
- 广告位管理

- 订单管理
- 退货管理
- 基础设置
- 后台用户管理

## 环境搭建与部署

### 开发环境

1. 通过在终端运行以下命令克隆或下载存储库:

   ```
   git clone https://github.com/lizesheng0313/FMM-Admin
   ```

2. 使用 pnpm 安装项目依赖项:
   ```
   pnpm install
   ```
3. 运行项目
   ```
   pnpm run dev
   ```
4. 生产环境部署
   ```
   pnpm run build
   ```
